{{HTMLSidebar("Global_attributes")}}

part は グローバル属性で、要素のパート名のスペース区切りのリストが含まれています。 CSS はパート名を使用して、 {{CSSxRef("::part")}} 擬似要素を介してシャドウツリー内の特定の要素の選択およびスタイルの設定を行うことができます。

使用例については Shadow part example をご覧ください。
