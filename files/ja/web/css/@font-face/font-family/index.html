{{CSSRef}}

font-family は CSS の記述子で、 {{cssxref("@font-face")}} 規則で指定されたフォントのフォントファミリーを指定することができます。

構文

/* <string> 値 */
font-family: "font family";
font-family: 'another font family';

/* <custom-ident> 値 */
font-family: examplefont;

値

<family-name>
フォントファミリーの名前を指定します。

公式定義

{{cssinfo}}

形式文法

{{csssyntax}}

例

フォントファミリー名の設定

@font-face {
  font-family: examplefont;
  src: url('examplefont.ttf');
}
