{{CSSRef}}

CSS の <flex> データ型 は、グリッドコンテナー内部における可変の長さです。 {{cssxref("grid-template-columns")}} や {{cssxref("grid-template-rows")}} 及びほかの関連プロパティで使われます。

構文

<flex> データ型は {{cssxref("<number>")}} のあとに単位 fr が付きます。すべての CSS dimensions と同様に、数値と単位の間に空白は入れません。

例

1fr    /* 整数値の使用 */
2.5fr  /* 浮動小数点値の使用 */
