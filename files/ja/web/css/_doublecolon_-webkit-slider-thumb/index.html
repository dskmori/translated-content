::-webkit-slider-thumb は CSS の擬似要素で、数値の値を変更する {{HTMLElement("input")}} の type="range" でユーザーが「溝」の中を動かすことができる「つまみ」を表します。

仕様書

どの仕様書でも定義されていません。これは WebKit/Blink に固有の私有擬似要素です。

ブラウザーの互換性
