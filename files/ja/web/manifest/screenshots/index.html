型 Object

screenshots メンバーはアプリケーションのショーケースに向けたスクリーンショットの配列を定義します。これらの画像はプログレッシブウェブアプリストアで使用することを意図しています。

例

"screenshots" : [
  {
    "src": "screenshot1.webp",
    "sizes": "1280x720",
    "type": "image/webp"
  },
  {
    "src": "screenshot2.webp",
    "sizes": "1280x720",
    "type": "image/webp"
  }
]
