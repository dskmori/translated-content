DOM の table column オブジェクト（ HTML の {{HTMLElement("col")}} 要素または {{HTMLElement("colgroup")}} 要素で示される範囲に相当） は、table column 要素を操作する為の特別なプロパティを持つ HTMLTableColElement インタフェースを公開します。
（※係るプロパティには、{{domxref("HTMLElement")}} オブジェクトインタフェースからも継承によりアクセス可能となっています）

プロパティ

{{domxref("HTMLElement")}} から継承するプロパティに加え、以下の固有プロパティを持ちます。

プロパティ名 型 説明
align {{obsoleteGeneric('inline','HTML5')}} {{domxref("DOMString")}} テーブル列内のセルのコンテンツの水平方向の配置位置を設定
ch {{obsoleteGeneric('inline','HTML5')}} {{domxref("DOMString")}} セルのデータの桁揃えの基準位置となる文字を設定
chOff {{obsoleteGeneric('inline','HTML5')}} {{domxref("DOMString")}} セルのデータの桁揃えの基準位置となる文字のオフセットを設定
span unsigned long {{htmlattrxref("span", "col")}} HTML 属性を反映。このオブジェクトの列の数を示す正の整数
vAlign {{obsoleteGeneric('inline','HTML5')}} {{domxref("DOMString")}} セル内のコンテンツの垂直位置を設定
width {{obsoleteGeneric('inline','HTML5')}} {{domxref("DOMString")}} 既定の列幅を設定

メソッド

固有のメソッドはありません。{{domxref("HTMLElement")}} から継承するメソッドを利用できます。
