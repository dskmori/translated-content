Le mot-clé modulepreload de l'attribut {{HTMLAttrxRef("rel", "link")}} de l'élément {{HTMLElement("link")}} fournit une manière déclarative d'analyser de façon préventive un module JavaScript et ses dépendances, ainsi que de les stocker dans la carte des modules du document pour une évaluation future.
