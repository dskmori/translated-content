Mathematical Markup Language (MathML) est un dialecte de XML permettant de décrire des formules mathématiques à la fois dans leur structure et dans leur contenu.

Vous trouverez ici des liens vers la documentation, les exemples et les outils permettant de travailler avec cette technologie. Pour un bref aperçu, veuillez consulter la présentation pour le Mozilla Summit Innovation Fair de 2013 (en).

Outils facilitant le développement avec MathML

  • Le validateur W3C
  • L'extension FireMath pour Firefox
  • L'ensemble d'extensions Mathzilla pour Firefox
  • TeXZilla - Convertisseur Javascript de LaTeX à MathML (demo, Firefox OS webapp, Firefox add-on, utilisation dans une page Web Page, un programme JS etc)
  • LaTeXML - Transformez vos documents LaTeX en pages HTML+MathML.
  • Web Equation - Transformez des équations manuscrites en MathML ou LaTeX
  • Mathjax - Moteur de rendu de Javascript pour les formules mathématiques, compatible avec tous les navigateurs. Pour forcer MathJax à utiliser le MathML natif, essayez cette extension pour Safari ou ce script GreaseMonkey.
