Materiały referencyjne technologii sieciowych

Web APIs
Mateiał referencyjny każdego z interfejsów zawierających się w Web APIs, włączając w to DOM i wszystkie powiązne API i interfejsy, które są używane do tworzenia stron WWW.
HTML
HyperText Markup Language jest językiem służącym do opisania i definicji zawartości stron internetowych. 
CSS
Cascading Style Sheets są używane do opisu wyglądu stron internetowych. 
SVG
Scalable Vector Graphics umożliwiają opis zdjęć jako zbiorów wektorów i figur. Sprawia to, że będą one zawsze dobrze wyglądać bez względu na rozmiar, w którym są prezentowane.
MathML
Mathematical Markup Language umożliwia pokazywanie skomplikowanych równań matematycznych i składni.
