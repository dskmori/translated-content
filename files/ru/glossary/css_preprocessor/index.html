CSS препроцессор (CSS preprocessor) - это программа, которая имеет свой собственный синтаксис ({{Glossary("syntax")}}), но может сгенерировать из него {{Glossary("CSS")}} код . Существует множество препроцессоров. Большинство из них расширяет возможности чистого CSS, добавляя такие опции как: примеси, вложенные правила, селекторы наследования и др. Эти особенности облегчают работу с CSS: упрощают чтение кода и его дальнейшую поддержку.

Для использования CSS препроцессора нужно установить CSS компилятор на ваш веб-сервер ({{Glossary("server")}}).
